# 佣书（Chronicle）

## 项目定位

历史互动叙事游戏。以屈大均北方游历为线索，以小见大，最终指向广州庚寅之劫的多重真相。

## 美术

- 宣纸质感，极简克制
- 背景色 `#F7F1E3`，文字 `#2F2A25`，强调 `#8C3A2B`
- 标题：霞鹜文楷；正文：思源宋体

## UI 原则

- 一个页面不要超过三个主要按钮
- 不要出现现代图标
- 更像数字博物馆，而非典型游戏 UI

## 剧情原则

- 没有真正的坏人
- 没有唯一的真相——所有人说真话，但所见不同
- 核心变量：对遗民、清廷、教士等的 **信任（Trust）**

## 技术原则

- 剧情全部写在 `src/story/*.json`
- 人类负责剧情与历史设定，AI 负责代码
- MVP 不做后端、不做账号、不做联网
